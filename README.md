# Maison Fleur – Parfüm Shop

Ein einfacher Mehrseiten-Website-Projekt für einen Parfümshop mit HTML, CSS und JavaScript.

## Inhalte
- Startseite mit Hero-Bereich und Bestseller
- Produktseite mit Duftübersicht
- Über-uns-Seite
- Kontaktseite mit einfachem Formular

## Struktur
- index.html – Startseite
- produkte.html – Produktübersicht
- ueber-uns.html – Unternehmensinfos
- kontakt.html – Kontaktformular
- styles.css – Gesamtgestaltung
- script.js – Navigation und Formularfunktion

## So startest du das Projekt
1. Öffne den Ordner in einem Browser oder starte einen lokalen Server.
2. Alternativ kannst du die Datei index.html direkt im Browser öffnen.

## Web-URL
- Lokal: http://localhost:8000/
- Oder direkt: file:///.../index.html

## Hinweise
Das Projekt ist bewusst einfach gehalten, damit es auch für einen Schüler leicht verständlich und erweiterbar ist.
