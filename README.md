# Maison Fleur Webshop

Maison Fleur ist ein Webshop für Parfümprodukte mit HTML, CSS und JavaScript ohne Frameworks. Das Projekt wurde gemäss dem M293-Auftrag als responsiver Mehrseitenauftritt umgesetzt.

## Funktionen

- Startseite mit beliebten Produkten, Kategorien und Newsletter-Anmeldung
- Produktseite mit allen Produkten und Kategoriefilter
- Detailseite mit Bild, Beschreibung, Preis sowie Merkliste- und Warenkorb-Funktion
- Kontaktseite mit CEO-Informationen und Formular inklusive Betreffauswahl
- Warenkorb mit `localStorage`
- Merkliste mit `localStorage`
- Einfaches Konto mit Login und Logout über `localStorage`

## Projektstruktur

- `index.html` Startseite
- `produkte.html` Produktübersicht
- `detail.html` Produktdetailseite
- `cart.html` Warenkorb
- `wishlist.html` Merkliste
- `account.html` Konto
- `ueber-uns.html` Hintergrundinformationen
- `kontakt.html` Kontaktseite
- `data.js` Produktdaten
- `script.js` Navigation, Filter, Web Storage API und Formularlogik
- `styles.css` Gestaltung und Responsive Design
- `wireframes.md` Wireframes für Mobile, Tablet und Desktop
- `styleguide.md` Farben, Typografie und Layoutregeln
- `KI-DOKU.md` Evaluation, Einsatz und Reflexion zur KI-Nutzung

## Web Storage API

Im Projekt werden die optionalen Funktionen lokal im Browser gespeichert:

- Warenkorb
- Merkliste
- Konto / Login-Status

Die Daten werden mit `localStorage` gespeichert und bleiben deshalb auch nach dem Neuladen der Seite erhalten.

## Lokaler Start

1. Projektordner öffnen.
2. `index.html` direkt im Browser starten oder einen lokalen Webserver verwenden.
